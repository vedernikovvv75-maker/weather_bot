# Currency Converter CLI

Небольшое CLI-приложение для получения курсов валют, кэширования ответа в JSON и конвертации сумм между валютами.

## Возможности

- Получение курсов через публичный API.
- Кэширование ответа в `currency_rate.json` на 24 часа.
- Вывод курсов для `RUB`, `EUR`, `GBP`.
- Конвертация суммы: `из валюты` -> `в валюту`.
- Валидация кодов валют с подсказкой доступных кодов.
- Понятные сообщения об ошибках без traceback в консоль.

## Структура

- `currency.py` — точка входа.
- `cli.py` — интерфейс командной строки и бизнес-логика конвертации.
- `api_client.py` — HTTP-клиент для запроса курсов.
- `storage.py` — чтение/запись кэша и проверка свежести.
- `requirements.txt` — зависимости.

## Быстрый старт

1. Создать и активировать виртуальное окружение:

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

2. Установить зависимости:

```powershell
python -m pip install -r requirements.txt
```

3. Запустить приложение:

```powershell
python .\currency.py
```

## Как работает кэш

- При первом запросе данные берутся из API и сохраняются в `currency_rate.json`.
- Если файл есть и ему меньше 24 часов, используются кэшированные данные.
- Если кэш отсутствует, устарел или поврежден, приложение запрашивает свежие данные.

## Пример сценария

1. Введите базовую валюту, например `USD`.
2. Приложение покажет курсы `RUB`, `EUR`, `GBP`.
3. Введите валюту источника, валюту назначения и сумму.
4. Получите результат в формате с 4 знаками после запятой.

Пример вывода конвертации:

```text
10.0000 USD = 755.0302 RUB
```

## Проверка типовых ошибок

- Неверный код валюты (например, `ZZZ`) -> выводится сообщение с доступными кодами.
- Сетевая ошибка/API ошибка -> выводится понятное сообщение.
- Проблема с файлом кэша -> выводится понятное сообщение, без stack trace.
