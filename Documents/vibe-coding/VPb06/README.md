# Telegram Tasks Bot (aiogram + SQLite3)

Минимально работающий бот для хранения задач в SQLite и их выгрузки в CSV.

## Команды

- `/start` — приветствие и подсказка по командам
- `/add` — добавить новую задачу (двухшагово)
- `/list` — вывести все задачи
- `/list_csv` — выгрузить задачи в CSV-файле

## Настройка

1. Заполните файл `.env` в корне проекта.
2. В `.env` должны быть:
   - `BOT_TOKEN`
   - `DATABASE_PATH` (по желанию, по умолчанию `tasks.db`)

Пример: см. файл `EnvExample`.

## Запуск

1. Установите зависимости:
   - `pip install -r requirements.txt`
2. Запустите:
   - `python main.py`

## Эксперименты (для проверки требований)

1. Добавьте несколько задач через `/add`.
2. Перезапустите бота и убедитесь, что задачи сохранились (SQLite persistence).
3. Проверьте, что `/list` показывает задачи в порядке добавления.
4. Выполните `/list_csv`, откройте CSV в Excel или LibreOffice.
5. Измените CSV: добавьте колонки `Статус` и `Категория`.

## Скриншоты (для сдачи)

Сделайте 3 скриншота:

- добавление задачи (`/add` + подтверждение)
- вывод списка (`/list`)
- выгрузка (`/list_csv` + отправленный документ)
